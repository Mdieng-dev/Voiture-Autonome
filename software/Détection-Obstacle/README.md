# Module Détection Ostacles

## Description
Détection Ostacles via capteur ultrason, Affichage distance par rapport à l’obstacle
Intégrer communication avec l'avertisseur sonore.

3 capteurs SRF10

## Sorties
Distance obstacle , signal sonore (variable)
